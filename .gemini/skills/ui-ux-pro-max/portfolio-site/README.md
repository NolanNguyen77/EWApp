# Professional Portfolio Site

A clean, minimalist, and performance-focused portfolio template designed with the **Swiss Modernism 2.0** style from the UI/UX Pro Max system.

## Features

- **Design**: "Swiss Modernism" meets "Classic Elegant" typography.
- **Tech**: HTML5 + Tailwind CSS (No build step required for simple edits).
- **Sections**:
  - Hero with animated entrance.
  - Skills grid.
  - Featured Projects.
  - Experience Timeline.
  - Contact Section.

## Customization

1.  **Open `index.html`** in any text editor (VS Code, Notepad, etc.).
2.  **Edit Content**:
    - Search for "Nolan Nguyen" to replace with your name.
    - Update the `href` in the **Projects** section to point to your actual GitHub repos or case studies.
    - Update the **Experience** section with details from your CV.
3.  **Images**:
    - The current images are placeholders using Lucide icons.
    - To use real screenshots, replace the `<div class="bg-zinc-100 ...">` blocks with `<img>` tags.

## Usage

Just double-click `index.html` to view it in your browser.
