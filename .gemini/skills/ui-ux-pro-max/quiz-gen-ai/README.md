# QuizGen AI - Frontend Prototype

This is a high-fidelity UI prototype for **QuizGen AI**, an educational platform that converts documents into multiple-choice quizzes.

## Features

- **Modern UI**: Built with Tailwind CSS, featuring Glassmorphism, smooth animations, and a professional educational color palette.
- **Interactive Flow**:
    1.  **Landing Page**: Hero section explaining the value proposition.
    2.  **Upload Interface**: Mock Drag & Drop zone for PDF/CSV files.
    3.  **Processing State**: Simulated AI loading state.
    4.  **Quiz Interface**: Full-screen overlay to take the generated detailed quiz.

## How to Run

Simply open `index.html` in any modern web browser.

> **Note**: This is a frontend prototype. The "Upload" feature is simulated for demonstration purposes. Clicking the upload area will trigger the "Analyzing" animation and then show the Quiz Interface after 2.5 seconds.

## Tech Stack

- **HTML5**
- **Tailwind CSS** (via CDN for portability)
- **Lucide Icons**
- **Google Fonts** (Poppins & Open Sans)
